Тестовое задание на позицию разработчика интерфейсов.

### Технологии

Angular 8, Webpack, SCSS

### Методологии

SCACSS, BEM

## Разработка

`yarn dev` для dev сервера. Приложение хостится по адресу `http://localhost:4200/`.

## Сборка

`yarn build` для сборки проекта.

## Деплой
